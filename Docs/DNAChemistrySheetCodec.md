# DNA chemistry sheet codec tranche

This tranche continues merged PRs #175 and #206 from architecture into a concrete, theorem-facing baseline.

## Implemented chain

```text
raw DNA
  -> exact U/V chart
  -> operational chemistry feature
  -> assay-relative quotient policy
  -> signed chemistry coordinates
  -> finite exact sheet-tower inhabitant
  -> exact chemistry transport ledger
  -> balanced ternary quantizer
  -> involutive phase-sector operator
  -> adaptive streaming-codec theorem surface
  -> versioned empirical validation boundary
```

## Operational quotient

`OperationalChemistryQuotient` no longer uses the lossless U/V pair as the quotient key. The feature contains coarse GC load, complement-span risk, reverse-complement risk, short-hairpin risk, aggregate thermo risk and an admissibility flag. Equality or a supplied tolerance policy on these features induces the operational basin relation.

The exact U/V chart remains available separately and remains reversible.

## Finite sheet tower

`FiniteChemistrySheetTower` is the first concrete inhabitant of `MultiscaleSheetTower`. It declares five DNA levels and exact reconstruction. The leaf contains the complete signed U/V signal; coarse levels are currently conservative typed slots. This closes the missing inhabitance problem without falsely claiming a particular averaging, wavelet or tetrational law.

A non-degenerate refinement must later populate the coarse levels and prove that it preserves the same signal and reconstruction semantics.

## Chemistry transport

`ChemistrySheetTransport` splits the current chemistry Hamiltonian into:

- exact local motif terms;
- signed sheet transition/coupling terms;
- an explicit residual slot.

For the current model the residual is zero because all existing terms are retained in the ledger. This is not a claim that all chemistry is band-local.

## Ternary and phase

The current signed coordinate carrier is already exactly three-valued, so its conversion to balanced trits is lossless. This gives a zero-distortion theorem only for the present discrete signed image. Quantizing a later real-valued tower still requires a separate distortion certificate.

The phase layer now has a concrete involutive sign-flip operator. Neutral is a fixed sector and positive/negative form a paired orbit. Full eigenspace promotion remains indexed by an operator-specific eigenlaw.

## Tetration

The physical DNA carrier remains `fixedFourWayNesting`. Tetration is assigned to a separate recursively nested transform/context semantics and cannot be inferred from carrier shapes. Promotion requires an explicit cardinal-growth theorem.

## Streaming codec

A direct trit/base codec is proved invertible. The production chemistry-aware codec is represented by `AdaptiveChoicePolicy`, which explicitly owns:

- 2/3/4/blocked choice arity;
- escape and extra-capacity actions;
- deterministic ordered choices;
- admissibility proofs;
- local decoding inverse;
- finite-state, checksum, recovery and payload-rate obligations.

## Empirical authority

Nearest-neighbour energies, mismatch and loop penalties, dimer risk and synthesis/sequencing risk are versioned parameters. Assay outcomes and the sufficient-statistic implication live in `EmpiricalValidationCertificate`; no wet-lab authority is manufactured by the formal layer.

## Remaining frontier

The highest-value next work is:

1. populate the coarse sheet levels with non-degenerate 4/16/64/256 summaries and prove reconstruction;
2. instantiate real nearest-neighbour and folding parameter tables with provenance;
3. construct a chemistry-admissible `AdaptiveChoicePolicy` and discharge its inverse/rate/checksum fields;
4. produce a distinct-sequence operational-quotient witness;
5. fit and validate the assay-relative tolerance policy on measured data;
6. prove a real-valued ternary distortion theorem if the tower is extended beyond signed discrete coordinates;
7. prove any claimed tetrational cardinal-growth law for transform contexts.
