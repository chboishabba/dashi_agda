# 2. The Cabibbo Parameter from the Disc -3 Lane

Let
\[
\alpha_1 = \sqrt{\frac{m_u}{m_c}}
\]
be the carrier's first mass-ratio coupling.  The relevant CM input for the
Cabibbo angle is the discriminant \(-3\) field
\(\mathbb{Q}(\sqrt{-3})\), with maximal order
\(\mathbb{Z}[\omega]\).  This is the Eisenstein lane.  Its modular level is
represented by \(X_0(3)\), whose standard hyperbolic volume is
\[
\mathrm{vol}(X_0(3)) = \frac{4\pi}{3}.
\]

The DASHI carrier normalises the disc \(-3\) modular volume against the CM
period scale of the Eisenstein fundamental domain.  Under this normalisation
the geometric multiplier is
\[
\pi\sqrt{3}.
\]
Equivalently, the multiplier may be viewed as the \(X_0(3)\) volume rescaled
by the hexagonal CM period unit.  The Cabibbo diagnostic is therefore
\[
\boxed{\lambda = \alpha_1 \pi\sqrt{3}}.
\]

With the current mass-ratio convention this gives
\[
\lambda_{\mathrm{carrier}} = 0.22441.
\]
The reference comparison value used in this draft is
\[
\lambda_{\mathrm{PDG}} = 0.22500,
\]
so the relative error is
\[
\frac{0.22441 - 0.22500}{0.22500} = -0.26\%.
\]

This is the strongest CKM result in the paper.  It uses no fitted CKM angle:
the only numerical input is the mass-ratio coupling \(\alpha_1\), while the
factor \(\pi\sqrt{3}\) is fixed by the disc \(-3\) CM geometry.  The result is
nevertheless still a Cabibbo-sector diagnostic rather than a full matrix
derivation.  In particular, it does not determine the CP phase or the
\((1,3)\) CKM entry.
