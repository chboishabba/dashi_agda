# Multi-Observer Science Quotient

`DASHI.Reasoning.MultiObserverScienceQuotient` records the thread tail where a
latent state carrier is only observed through many-to-one quotients.  Apparent
randomness is represented as ambiguity at the observation surface, and science
is represented as multi-observer quotient fusion followed by residual and
admissibility checks.

The safe pipeline is:

```text
latent carrier, when supplied
  -> observer-dependent quotient
  -> apparent outcome ambiguity
  -> multi-observer slice fusion
  -> peer-review / replication residual checks
  -> candidate reconstruction
```

The module is deliberately non-promoting.  It does not prove universal
determinism, hidden variables, scientific truth closure, peer-review
infallibility, full latent-state recovery, or an observer-neutral complete
slice.
