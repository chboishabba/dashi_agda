# Inverse Bidirectional Culture Operators

`DASHI.Culture.InverseBidirectionalCultureOperators` records fiction,
nonfiction, reading aloud, audience, child-context, art, architecture, poetry,
and politics as bounded inverse/bidirectional cultural manifold operators.

The safe reading is:

```text
cultural artifact / performance / built form
  -> imagined or claimed world model
  -> body / attention / social expectation update
  -> feedback into reader, listener, audience, or population
  -> revised shared manifold
```

The WAIS/autism-adjacent material is kept only as an observation quotient over
pruning/search styles.  It is not a clinical diagnosis, not WAIS interpretation
authority, and not a claim that a bounded test recovers cognition.

The blocked claims are clinical autism diagnosis, WAIS interpretation
authority, culture determinism, propaganda efficacy theorem, qualia recovery,
child development closure, and aesthetic truth.

`DASHI.Culture.KnotWeaveTopologyCultureBridge` extends this lane with
knot/weave/braid as bounded semantic/topological operator carriers.  That
bridge reuses the inverse-culture surface but adds no cultural universal,
identity closure, qualia closure, or authority promotion.
