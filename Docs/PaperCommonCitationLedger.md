## Usage rules

1. Use these keys verbatim in the live drafts unless a paper-specific reason
   requires an extension.
2. If a citation is imported only as background or scope control, say so in the
   prose.
3. If a citation carries an external theorem step, the claim-boundary table of
   the consuming paper must list that step under "assumed externally with
   citation".
