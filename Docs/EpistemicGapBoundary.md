# Epistemic Gap Boundary

This module formalises a narrow rule:

> An absence of explanation is not evidence for a particular explanation.

The rule is intentionally compatible with spiritual, Indigenous, existential,
mathematical, Buddhist, and Spinozist meaning-making. It does not reduce those
frames to error. Instead, it assigns interpretation and causal explanation
separate types.

## Core types

- `EvidenceState A`
  - `observed a`
  - `derived a`
  - `postulated`
  - `unknown`
- `Grounded state`
  - constructible only for `observed` and `derived`
- `ExplanatoryClaim`
  - a phenomenon, proposed explanation, and evidence state
- `WarrantedClaim`
  - an explanatory claim plus a `Grounded` witness
- `InterpretiveFrame`
  - a phenomenon, meaning, and meaning mode
- `ResearchObligation`
  - records that an unknown opens a question rather than closing it

## Minimal theorems

`unknownIsNotGrounded` and `postulateIsNotGrounded` are empty-pattern proofs.
No constructor can witness groundedness for either state.

`noUnknownPromotion` states that an explanatory claim whose evidence is
`unknown` cannot carry the warrant required for promotion.

`noPostulatePromotion` gives the same boundary for a merely postulated answer.

## Relation to “God of the gaps”

The formal target is not theology. It is an invalid promotion rule:

```text
unknown mechanism
therefore
specific metaphysical mechanism
```

No such function is present. A metaphysical or spiritual interpretation remains
representable as an `InterpretiveFrame`, but it becomes a warranted causal claim
only when an explicit evidence-bearing bridge is supplied.

This also protects non-theistic and Indigenous interpretations from being
misclassified as failed laboratory hypotheses. Relational meaning, immanence,
contemplative practice, or existential value can be represented without claiming
to fill a mechanistic gap.

## Origin-of-life example

`EpistemicGapBoundaryTests.agda` contains:

- observed and derived natural-mechanism claims;
- relational and Spinozist interpretations;
- an unknown origin claim that cannot be promoted;
- an open research obligation generated by the unknown.

## Claim boundary

This module proves only a typing boundary. It does not prove that every observed
or derived explanation is scientifically correct, nor that interpretation is
causally inert. Stronger reliability, replication, model-selection, and bridge
criteria must be added as separate structures.
