# GR / Quantum Proof-Term Upgrade

## Purpose

This tranche upgrades the merged GR/quantum unification facade from a theorem
**manifest** toward a theorem **implementation**.

The governing distinction is:

```text
Bool or Set-valued theorem name
  !=
inhabited proposition-level proof
```

`DASHI.Unified.GRQuantumProofTerms` therefore separates each theorem surface
from a corresponding inhabited `Closed` record.  The terminal object is now
`TerminalGRQuantumProof`; no canonical inhabitant is defined.

## Exact dependency route

```text
Multiscale residual reconstruction
  -> finite orthogonal energy uniqueness
  -> finite chain/antichain causal geometry
  -> constructive Clifford syntax and even grading
  -> finite tree/Weyl noncommutativity and Born weights
  -> exact flat Lorentzian/Einstein/source model
  -> exact trivial flat constraint algebra
  -> finite regional spectral enumeration
  -> conditional continuum and empirical authorities
  -> TerminalGRQuantumProof
```

The route deliberately distinguishes exact finite/model results from general
continuum or physical claims.

## Exact finite proofs added

### 1. Quadratic energy uniqueness

`DASHI.Core.FiniteQuadraticMultiscale` defines a four-state orthogonal carrier:

- zero;
- one coarse unit;
- one detail unit;
- their combined state.

The laws are:

```text
E(0) = 0
E(coarse) = 1
E(detail) = 1
E(combined) = E(coarse) + E(detail)
```

Pattern matching proves that every table satisfying these laws equals the
canonical quadratic table `(0,1,1,2)`.

This is an actual uniqueness proof.  The extension to arbitrary complete linear
or Hilbert carriers still requires the general parallelogram-law
characterization.

### 2. Chain and antichain geometry

`DASHI.Geometry.FiniteCausalDiamond` constructs:

- one past event;
- three mutually incomparable spatial events;
- one future event;
- explicit causal chains;
- a three-element separating antichain;
- exact chain-depth and antichain-width functions;
- cyclic spatial relabellings;
- invariance of both observables under those relabellings;
- adjacent null links and an explicit finite causal cone;
- exact finite `3 + 1` dimension counts.

`DASHI.Geometry.FiniteCausalDiamondClosed` supplies the inhabited closure
record.

This proves the finite model.  It does not yet prove that a large locally finite
poset or continuum fixed point uniquely has three spatial dimensions.

### 3. Constructive Clifford syntax

The quotient/setoid backend, universal factorization theorem, twisted-adjoint

### 4. Finite tree/Weyl and Born lane
