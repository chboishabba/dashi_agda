That gives:


# 5. Profile-wise combination theorem

Once all four profile bounds exist:

[
R_N^{p}C_N^{p}\le C_p^2N^{-2}
]

for:

[
p\in{FT,adv,trans,res}.
]

You need to combine:

[
K_N=\sum_pK_N^p.
]

For nonnegative kernels:

[
R_N
===

\sup_i \sum_j K_N(i,j)\frac{w_j}{w_i}
\le
\sum_p R_N^p.
]

Similarly:

[
C_N\le\sum_p C_N^p.
]

But from product bounds alone you do **not** automatically get separate row and column bounds. So the clean profile-wise theorem should store row and column witnesses, not only products.

For each profile (p), carry:

[
R_N^p\le A_pN^{-\alpha_p},
]

[
C_N^p\le B_pN^{-\beta_p},
]

[
\alpha_p+\beta_p\ge2.
]

Then:

[
R_N\le \sum_p A_pN^{-\alpha_p},
]

[
C_N\le \sum_p B_pN^{-\beta_p}.
]

However, if some profiles are forced-tail-like:

[
R_N^{FT}\le A_{FT},\quad C_N^{FT}\le B_{FT}N^{-2},
]

and adversarial-like:

[
R_N^{adv}\le A_{adv}N^{-2},\quad C_N^{adv}\le B_{adv}N^{-1},
]

then total row may be (O(1)), total column may be (O(N^{-1})), which gives only:

[
R_NC_N\lesssim N^{-1}.
]

So for the **global Schur product** you need more than profile product bounds unless the weights/profiles are arranged so that the row-heavy and column-heavy profiles do not stack badly.

There are three possible solutions.

## Combination Solution 1 — common orientation

Prove all profiles satisfy:

[
R_N^p\le A_p,
\qquad
C_N^p\le B_pN^{-2}.
]

Then:

[
R_N\le A,
\qquad
C_N\le BN^{-2},
]

so:

[
R_NC_N\le ABN^{-2}.
]

This is simple but may be too strong.

## Combination Solution 2 — split Schur vector weights by profile

Use profile-specific weights (w^p), then prove compatibility:

[
w^p\sim w
]

uniformly. If each profile closes in its best orientation under a compatible weight, then combine via equivalent weighted norms.

The theorem shape:

```agda
profileWeightCompatibility :
  ∀ p -> EquivalentWeights wProfile[p] wGlobal
```

Then:

```agda
profileWiseWeightedSchurCombination :
  ∀ p -> WeightedProductBound p wProfile[p] ->
  profileWeightCompatibility ->
  GlobalWeightedSchurProductBound wGlobal
```

## Combination Solution 3 — row/column vector product, not scalar product

Let:

[
r_p=R_N^p,\qquad c_p=C_N^p.
]

Then:

[
R_NC_N
\le
\left(\sum_pr_p\right)
\left(\sum_pc_p\right).
]

Expanding:

# [

\sum_{p,q}r_pc_q.
]

Profile product bounds only control (r_pc_p), not cross terms (r_pc_q).

So you need cross-profile compatibility:

[
\boxed{
r_pc_q\le C_{pq}N^{-2}
\quad\forall p,q.
}
]

This is the most honest general theorem.

The global combination target becomes:

```agda
profileCrossProductBounds :
  ∀ p q -> RowBound p * ColumnBound q <= C[p,q] / N²
```

Then:

[
R_NC_N
\le
\sum_{p,q}C_{pq}N^{-2}
======================

CN^{-2}.
]

This is probably the right DASHI theorem.

It means Stage 3 does not merely need four diagonal profile products. It needs either:

```text
all profiles aligned in same row/column orientation
```

or:

```text
cross-profile product matrix closes.
```

This is important.

# 6. Global weighted Schur product theorem

The actual global theorem should be stated as:

[
\boxed{
R_N(w_N)C_N(w_N)\le C^2N^{-2}.
}
]

where:

[
R_N(w_N)=\sup_i\frac1{w_i}\sum_jK_N(i,j)w_j,
]

[
C_N(w_N)=\sup_j\frac1{w_j}\sum_iK_N(i,j)w_i.
]

With profile decomposition:

[
K_N=\sum_{p\in P}K_N^p.
]

You can prove:

[
R_N\le\sum_pR_N^p,
\qquad
C_N\le\sum_pC_N^p.
]

Then either prove common orientation or cross-matrix closure.

The strongest general form:

[
\boxed{
\forall p,q,\quad R_N^pC_N^q\le C_{pq}N^{-2}.
}
]

Then:

[
R_NC_N
\le
\sum_{p,q}R_N^pC_N^q
\le
\sum_{p,q}C_{pq}N^{-2}
======================

C^2N^{-2}.
]

This should become:

```agda
profileCrossSchurMatrixBound
globalWeightedSchurProductFromProfileMatrix
weightedSchurProductBoundClosed
```

This avoids the false inference:

[
R_pC_p\le N^{-2};\forall p
\not\Rightarrow
(\sum R_p)(\sum C_p)\le N^{-2}.
]

That is the key mathematical caveat.

Then:

[
|\langle L_Nx,x\rangle|
\le

[

then:

[
|L_N|\le C/N
\Rightarrow
q_{\mathrm{gap}}\ge c/N^2.

But this needs an exact spectral relation.

Mathematical target:
